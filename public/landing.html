Fitur Utama

Semua yang tim Anda butuhkan dalam satu platform

📤

Excel Auto-Parse

Upload file rincian faktur penjualan Ayotama. Ribuan transaksi terekam otomatis. Outlet baru fuzzy-matched. Preview sebelum import.

📱

Mobile PWA

Install di home screen. Bisa akses offline. Pull-to-refresh. Semua salesman butuh HP saja — tidak perlu laptop atau app store.

💰

3-Tier Bonus Engine

Percentage (target BE), Volume (tier BE), Active Outlets (aktivasi). Dihitung otomatis, real-time, transparan untuk semua level (L2/L3).

🏥

Outlet Health Score (OHS)

Algoritma 3-bulan: 40% volume + 40% trend + 20% frekuensi. Skor 0-100. Outlet di bawah 40 otomatis di-flag. Tidak ada lagi churn tanpa disadari.

📊

SKU Analytics

Ranking SKU, MoM trend, YoY trend, weekly breakdown, top outlet per SKU, kontribusi outlet, dan mix percentage. Semua bisa drill-down.

👥

Team Dashboard

Supervisor lihat ranking, attainment, bonus, vacant outlet. Admin manage user, outlet, assignment, target — semua dari web panel.
